This web application uses Tesseract-OCR and the retrained tr-OCR model!
Instructions for installing Tesseract-OCR:
1. Download Tesseract-OCR:
https://github.com/UB-Mannheim/tesseract/wiki
2. Install it.
3. During installation, select English or other language packs.
4. Add the path to tesseract.exe to the PATH environment.

Instructions for installing tr-OCR:
1. Download the archive from my Google Drive: 
https://drive.google.com/file/d/15VhqJTglJyieJ0rITCE02lSc4GCQRMN6/view?usp=sharing
2. Unzip it to the "Program Files" folder

Даний веб-додаток використовує Tesseract-OCR та перенавчену модель tr-OCR!
Інструкція по встановленню Tesseract-OCR:
1. Завантажте Tesseract-OCR:
   https://github.com/UB-Mannheim/tesseract/wiki
2. Встановити його.
3. Під час встановлення обрати англійський або інші мовні пакети.
4. Додати шлях до tesseract.exe у середовище PATH.

Інструкція по встановленню tr-OCR:
1. Завантажте архів з мого гугл-диску: 
   https://drive.google.com/file/d/15VhqJTglJyieJ0rITCE02lSc4GCQRMN6/view?usp=sharing
2. Розпакуйте його в папку "Program Files"
